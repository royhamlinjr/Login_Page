> A modern, responsive **Login Form UI** built with **HTML5 and CSS3**.
> Features a glassmorphism-style design, clean Poppins typography, and a polished user experience suitable for any web authentication page.

---

## 🚀 Key Features

| Feature | Description |
|---|---|
| 🪟 **Glassmorphism UI** | Blur and transparency effects for a modern look |
| 📱 **Responsive Layout** | Fully centered and works across all screen sizes |
| 🔤 **Clean Typography** | Poppins font from Google Fonts |
| 🔐 **Form Elements** | Username, password, remember me, forgot password |
| 🎯 **HTML5 Validation** | Built-in `required` field validation |
| 🖱️ **Hover States** | Interactive button and link hover effects |
| ⚡ **No Dependencies** | Pure HTML & CSS — no frameworks or build tools |

## ▶️ Run Locally

> **No dependencies or build tools required**

### 1️⃣ Clone Repository
```bash
git clone https://github.com/royhamlinjr/Login_Page.git
cd Login_Page
```

### 2️⃣ Open in Browser
Simply open `index.html` in any modern web browser.

> Optionally replace `download.jpg` with your preferred background image.

## 🧠 Concepts Demonstrated

**CSS & Layout**
- Glassmorphism effect using `backdrop-filter` and transparency
- Flexbox for centered responsive layout
- Smooth hover transitions on buttons and links
- Rounded inputs and shadow effects

**HTML**
- Semantic HTML5 structure
- Built-in form validation using `required`
- Accessible input labels and placeholders
